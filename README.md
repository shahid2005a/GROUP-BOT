## 🎯 Commands

### 👤 User Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot |
| `/help` | Show help guide |
| `/balance` | Check your credits |
| `/buy` | Buy more credits |
| `/num 1234567890` | Search mobile number |
| `/adhar 123456789012` | Search Aadhaar |
| `/veh 1234567890` | Search vehicle |
| `/family 123456789012` | Search family members |
| `/upi name@paytm` | Search UPI ID |
